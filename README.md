# The Quantum One

High-end, one-page Flutter dashboard with an ultra-dark glassmorphism aesthetic.

## Features

- Background color `#020205` with frosted glass cards.
- Reusable `CustomGlassCard` with blur (`sigma=20`) and 1.5px light-catcher border.
- `ClimatePulseWidget` with local weather and global extremes toggle.
- `IntelBriefingWidget` with credibility score rings.
- `LifestyleOSWidget` with macro pie chart and wearable recovery score.
- `QuantumWisdom` quote footer in elegant serif italics.
- `Night Mode` toggle that shows only Health Performance + Quote.

## Run

1. `flutter pub get`
2. `flutter run`
